Website Evaluations

Niall Horan's website

This website is for the artist, Niall Horan. This site contains an overview of his recent music, concert dates, links to buy merch, and an area to sign up for his newsletter

    Contrast: This website has great contrast between the elements and are very appealing to the eye
    Repitition: This website has a very cohesive design, there is repition within the different sections of the website as you scroll through
    Aligment: The margins and padding on all sides of the website are consistent, and all text is cohesively aligned
    Proximity: All elements that go well together are grouped together, and are placed very intentionally

Yale's Art Department

This website is for yale's school of art. This site contains all the recent happenings within the department, upcoming events, and different tabs giving information about the school

    Contrast: This website is so incredibly chaotic. The colors and fonts do not mesh well together and it is very abrasive to the eye.
    Repitition: There is no consistency in this website, every element is different and does not go well together
    Alignment: All of the elements are mis-aligned and cause the viewer lots of confusion
    Proximity: This site does not have any steady grouping for any of it's elements. Everything is all over the place leading to a very difficult browsing experience.
